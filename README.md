# Site Pessoal - Currículo Online

Trabalho final da disciplina de Frontend (HTML, CSS e JavaScript).


## Sobre o projeto

A ideia era criar um site pessoal que funcionasse como currículo online, seguindo as diretrizes do site Infojobs.

Antes de migrar meus estudos pra area atual, eu mexia bastante com front-end. Fiz alguns cursos na área, incluindo React e CSS. Também fiz o curso Full Stack Developer do Rodolfo Mori (DevClub), que me ajudou bastante com organização de código e boas práticas. Resgatei um pouquinho dessa base pra desenvolver esse projeto.


## O que foi usado em termos de codigo

### HTML
- Estrutura semântica básica (`header`, `nav`, `main`, `section`, `footer`)
- Separação do conteúdo por seções (dados pessoais, formação, experiencia etc.)
- Navegação interna entre as seções
- Comentários indicando onde substituir pelos dados pessoais

### CSS
- Variáveis de cores pra manter consistência visual
- Flexbox e Grid pra organizar os elementos
- Design responsivo com `@media` (funciona no celular e no computador)
- Efeitos simples de `hover` nos cards
- Classes reutilizáveis (`.container`, `.card`, `.info-grid` etc.)
- Botão "voltar ao topo" estilizado

### JavaScript
- Função pra mostrar/esconder o botão de voltar ao topo conforme o scroll
- Rolagem suave ao clicar nos links
